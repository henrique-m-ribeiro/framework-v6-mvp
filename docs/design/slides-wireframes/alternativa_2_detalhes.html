Alternativa 2: Detalhamento

Componentes Principais

Navegação por Âncoras
Menu fixo no topo com links para cada dimensão. Scroll suave ao clicar. Indicador visual da seção ativa (underline cyan).
Estrutura de Seções Dimensionais
Cada dimensão ocupa altura completa da tela (viewport). Contém: título da dimensão, 3-4 KPIs, 2 gráficos, caixa de análise IA, tabela de dados.
Chat Flutuante Expansível
Botão circular fixo no canto inferior direito (60px). Ao clicar, expande para painel 400x600px sobrepondo conteúdo. Badge de notificação quando IA responde.
Menu de Ações Rápidas
Barra flutuante à esquerda com ícones: Exportar PDF, Exportar CSV, Comparar territórios, Alterar período, Ajuda.

Funcionalidades

Exportação Contextual
PDF: todas dimensões ou seção visível. CSV: dados da seção atual.
Scroll Suave Inteligente
Snap automático para início de cada seção. Indicador de progresso lateral.
Chat Overlay Responsivo
Chat se adapta ao tamanho da tela. Mobile: fullscreen. Desktop: painel flutuante.
Fluxo de Navegação
1
Selecionar território no header
2
Scroll pela página para ver todas dimensões
3
Ler análises IA geradas em cada seção
4
Clicar no chat flutuante para perguntas
5
Usar menu âncoras para navegação rápida
6
Exportar via ações rápidas laterais

Alternativa 1: Layout em Abas

Opção 1

Interface organizada em abas horizontais, onde cada aba representa uma dimensão de análise. Chat com IA permanece fixo na lateral direita, sempre acessível durante a navegação.

Abas horizontais: Uma aba para cada dimensão (Econômica, Social, Territorial, Ambiental)
Chat fixo lateral: Painel de IA sempre visível à direita (30% da tela)
Painel de controle: Seletores de território e período no topo
Navegação simples: Um clique para alternar entre dimensões
Exportação contextual: Botões de PDF/CSV visíveis em cada aba
